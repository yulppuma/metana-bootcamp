# metana-bootcamp Module-6
## Prerequisites
### Before you begin, ensure you are able to run the following tools on your system:
### Node.js & Node Version Manager (Node.js v16+ recommended) (https://nodejs.org/en)
### Git (https://git-scm.com/)
### VS Code (https://code.visualstudio.com/)
### Hardhat (installed via npm) (https://hardhat.org/)

## Setup for Windows via VS Code terminal
### 1. Clone the repository
``` git clone git@github.com:yulppuma/metana-bootcamp.git```
### 2. Install dependencies & swap to usable nodejs version
``` cd module-6```
``` cd CaptureTheEtherHardhat```
``` nvm use 18```
```yarn install```
### 3. Test the challenges with the corresponding file name. Ex:
``` npx hardhat test .\test\04-GuessTheNumber.test.ts```
